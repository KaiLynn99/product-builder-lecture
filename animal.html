나는 강아지상일까 고양이상일까?

정면 사진을 올리면 모델이 강아지상과 고양이상 확률을 계산합니다.

사진을 선택하면 분석을 시작합니다.

테스트가 작동하는 방식

이 페이지는 사용자가 선택한 이미지를 브라우저에서 읽고, Teachable Machine 이미지 분류 모델을 이용해 두 라벨의 확률을 계산합니다. 현재 모델의 라벨은 강아지와 고양이이며, 결과는 가장 높은 확률을 기준으로 표시됩니다.

  • 사진은 분석을 위해 사이트 서버로 업로드하지 않습니다.
  • 얼굴이 중앙에 있고 밝은 사진일수록 결과가 안정적입니다.
  • 모델은 재미용 분류 도구이며 사람의 성격, 매력, 외모를 단정하지 않습니다.

더 좋은 결과를 위한 사진 팁

모델은 이미지 안의 패턴을 기준으로 판단합니다. 흐릿한 사진, 여러 명이 함께 나온 사진, 얼굴 일부가 가려진 사진은 결과가 흔들릴 수 있습니다.

  1. 정면에 가까운 사진을 선택합니다.
  2. 얼굴 주변이 너무 어둡거나 과하게 밝지 않은 이미지를 사용합니다.
  3. 선글라스, 마스크, 강한 필터가 적용된 사진은 피합니다.
