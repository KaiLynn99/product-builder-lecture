테스트 전 알아둘 점

동물상 테스트는 이미지 분류 모델을 이용해 사진에서 강아지상과 고양이상에 가까운 패턴을 비교하는 재미용 도구입니다. 결과는 사용자가 올린 사진의 조명, 각도, 표정, 배경에 따라 달라질 수 있습니다.

사용 방법

  1. 동물상 테스트 페이지에서 사진 선택 버튼을 누릅니다.
  2. 얼굴이 잘 보이는 이미지를 선택합니다.
  3. 분석이 끝나면 강아지상과 고양이상 확률을 확인합니다.

결과 해석

확률이 높게 나온 라벨은 모델이 해당 이미지에서 더 강하게 감지한 패턴입니다. 특정 사람의 성격이나 실제 이미지를 판단하는 기준이 아니며, 같은 사람도 사진 조건이 바뀌면 결과가 달라질 수 있습니다.

사진 선택 팁

  • 정면 또는 정면에 가까운 사진을 사용합니다.
  • 얼굴이 화면 중앙에 있고 배경이 복잡하지 않은 사진이 좋습니다.
  • 마스크, 선글라스, 과한 필터가 있으면 결과가 부정확할 수 있습니다.
